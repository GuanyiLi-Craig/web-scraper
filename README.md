# web-scraper
web scraper practice

## Scrapy

* Install 
```bash
pip install scrapy
```

* Generate Project
```bash
scrapy startproject ProjectName # Example: scrapy startproject addgeneMining
```

* Generate Spider
```bash
scrapy genspider SpiderName Domain # Example: scrapy gensipder addgene www.addgene.org
```
